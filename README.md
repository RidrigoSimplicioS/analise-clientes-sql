#  Análise de Clientes com SQL

##  Objetivo

Este projeto tem como objetivo analisar uma base de dados de clientes para identificar padrões de comportamento, problemas de retenção e oportunidades de aumento de receita.

---

##  Tecnologias utilizadas

- MySQL
- SQL

---

## 📁 Estrutura da base de dados

A base contém informações como:

- Nome
- Idade
- Cidade
- Profissão
- Salário
- Gasto mensal
- Frequência de compra
- Último acesso
- Status (ativo/inativo)

---

##  Análises realizadas

### 🔴 Taxa de clientes inativos

- 51% dos clientes estão inativos (102 clientes)

 **Insight:**
Alta taxa de inatividade indica risco de perda de receita.

---

### 🟡 Clientes com alto potencial

- Clientes com salário acima de R$6000 e baixo consumo

📌 **Insight:**
Existe oportunidade de aumentar o faturamento com estratégias direcionadas.

---

### 🔵 Comportamento de compra

- Clientes com baixa frequência de compra

 **Insight:**
Baixo engajamento pode impactar diretamente a receita.

---

##  Recomendações

- Criar campanhas de reativação
- Desenvolver ofertas personalizadas
- Melhorar estratégias de fidelização
- Utilizar automação para comunicação com clientes

---

##  Possível evolução

Este projeto pode ser integrado com um bot de WhatsApp para reengajamento automático de clientes.
